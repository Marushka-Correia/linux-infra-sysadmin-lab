# Linux Infra System Admin Lab 

This repository documents my **hands-on Linux system administration practice**, starting from absolute beginner fundamentals and progressing 
toward **real-world infrastructure and sysadmin tasks**.

The goal of this lab is to build **practical Linux confidence** .

All work is performed on an **Ubuntu Server (LTS)** environment using a virtual machine, simulating real production server behavior.

---

##  Project Objectives

Build strong Linux fundamentals from scratch for beginners

Help learners understand how Linux servers are structured and managed

Provide hands-on practice with commands used daily by system administrators

Encourage troubleshooting skills and infrastructure thinking through real examples
